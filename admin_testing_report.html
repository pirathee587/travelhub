7. Testing & Evaluation

7.1 Testing Approach

This chapter presents the functional testing conducted on the TravelHub Admin Panel. Black-box testing methodology was used throughout, focusing on the expected behaviour of each module from an end-user perspective. Every identified test case was executed against the live system and the actual results were compared with the expected results. All tests listed in this document resulted in a Pass.

The following sections (7.2.1 – 7.2.7) cover: Admin Dashboard, Hotel Approvals, Agent Approvals, Package Approvals, Payments Management, Analytics, and Admin Profile Management.
