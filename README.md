# Vehicle Management Agent

## Project info

This is a vehicle management application built with React, Vite, and Tailwind CSS.

## Getting Started

To get started with development:

1.  **Clone the repository**
    ```sh
    git clone <YOUR_GIT_URL>
    ```

2.  **Install dependencies**
    ```sh
    npm install
    ```

3.  **Start development server**
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
